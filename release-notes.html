Release Notes

Public changes and verification status for the current release candidate.

Core suite

12 bilingual, single-file applets covering search, logic, probability, machine learning, vision, and reinforcement learning.

Instructional layer

Five scenarios per applet, a featured comparison, a visual explanation, worksheet prompts, student response packet, and text-state accessibility support.

Teacher materials

Printable Teacher Pack, responsive Curriculum Map, one-page Student Lab Packet, public Quality Notes, and release notes.

Release preparation

Public routing, print layouts, packet rendering, responsive controls, section order, scenario behavior, author identity, and QA gates were hardened before launch media.

Verification status
  • Static release checker gates public structure and known regressions.
  • Playwright browser QA tests all public pages across three viewports.
  • Print and physical-phone checks remain manual release gates.
Evidence limits

This release does not claim learning gains, cognitive-load reduction, adoption, WCAG conformance, or public uptake. Those outcomes require separate evidence.
